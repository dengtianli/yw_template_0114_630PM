下拉菜单

jquery select2插件

简介

-----select2是jquery扩展的美化下拉框的插件，它能让简单的select下拉框变得更好看、更方便，支持搜索，远程数据集以及滚动的效果。它提供给了核心js包select2.js和样式文件select2.css，使用时引入他们即可，同时别忘了先引入jquery。

使用方法

-----最简单的使用方式：在页面加载完成后调用$("#selectId").select2();

<script src="js/jquery-1.11.1.min.js"></script> <script src="js/select2.min.js"></script> <link rel="stylesheet" type="text/css" href="css/select2.css" /> <select class="full_width" id="select1"> <option value=""></option> <option value="选项1">选项1</option> <option value="选项2">选项2</option> </select> <script> jQuery(document).ready(function() { $("#select1").select2(); }); </script>

-----禁用选项：在选项option中添加属性disabled="disabled"。

-----多选：只要定义属性multiple=""就行。

-----动态绑定数据：可以采用select2提供的ajax方式获取数据组织html。
